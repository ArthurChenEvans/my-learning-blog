String Manipulation

Created: 30th August 2025 | Last Updated: 30th August 2025

Basics of Strings

Strings in Javascript, along with numbers, booleans, bigInts, and more are what are known as primitive types. Primitive types involve one 'type' of value and not collections of various types like reference types (think objects, maps, arrays, etc). A key property regarding primitive types is that they are immutable, which means the values themselves are never modified in memory once created. So what do you do if you want to edit said value, such as removing whitespace from a string, or padding it with some characters, or replacing values if the value itself can never be modified? To get around this, we temporarily wrap the primitive type in a wrapper object that accesses the string. This wrapper object allows for methods to be called on it, which builds a new string by applying a method's logic on the primitive value. This process is known as boxing.

String Querying

To begin with, I'll start with methods that involve basic string querying. This would involve checking the first and last index of an element, whether the string includes a given argument or not, and whether a string starts with or ends with an argument. I believe code will be simpler to explain than, well, trying to explain them since the keywords for the methods used are clear and explicit anyway:


const my_string = "Hello World!";

// Methods find the first and last index of a given substring respectively
my_string.indexOf("o"); // 4
my_string.lastIndexOf("o"); // 7

// The following 3 methods check whether a substring exists at all, or whether the string starts or ends with it
my_string.includes("llo"); // true
my_string.includes("llo"); // true
my_string.includes("He"); // true

// While not solely querying, these two methods do involve searching for a substring and then replacing it
my_string.replace("!", "?"); // "Hello World?"
my_string.replaceAll("o", "0"); // "Hell0 W0rld?"

While not used to query for a specific substring within a string, there is a useful property to check for the number of characters in a string (or the number of elements in an array!). This is done via my_string.length;.

String Editing

This is where the bulk of the methods come in. I'll split them into two groups: simple edits and whole modifications. Simple involves trimming, padding, and repeating, while whole modifications involve extracting a substring, separating a string into a list of strings, and combining a list of strings into a string:


const my_string = "   Hello World!   ";

// Simple modifications: trims
my_string.trim(); // "Hello World!"
my_string.trimEnd(); // "   Hello World!"
my_string.trimStart(); // "Hello World!   "

// Simple modifications: paddings
const pad_string = "I'm padded!";
pad_string.padStart(3, "+"); // "+++I'm padded!"
pad_string.padEnd(3, "+"); // "I'm padded!+++"

// Simple modifications: repeat
const repeat_string = "repeat.";
repeat_string.repeat(3); // "repeat.repeat.repeat."

// Whole modifications: slicing
const slice_string = "String To Slice!";
slice_string.slice(7, 9); // "To"
slice_string.slice(7); // "To Slice!" - Gets to end of string if only start value given

// Whole modifications: splitting
const split_string = "String To Split";
split_string.split(" "); // ["String", "To", "Split"]

// Whole modifications: joining
const join_array = ["array", "to", "join"];;
join_array.join("+"); // "array+to+join"
