What is a flexbox?

A flexbox is simply a type of container that allows for more granular position of its child items using an x/y axis for alignment. On top of item alignment, it also takes care of spacing, gaps, item size distribution based on available space, and how items wrap around based on available space.

Take for instance the following: 


.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
